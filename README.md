#  BrainTrack – Smart Study Assistant Website

##  Project Overview

BrainTrack is a web-based smart study assistant designed to help students stay organized, focused, and productive. The platform combines task planning, time management, and progress tracking into one simple and interactive website.

This project addresses common challenges students face, such as procrastination, poor time management, and difficulty prioritizing tasks.

##  Features

### Smart Task Planner
- Add tasks with subjects, descriptions, and deadlines
- Automatic organization by urgency (overdue, urgent, soon, future)
- Mark tasks as completed
- Delete tasks

### Smart Suggestion System
- "What Should I Study Now?" button
- Intelligently recommends the most urgent task
- Reduces decision fatigue

### ⏳ Focus Timer (Pomodoro Technique)
- 25-minute study sessions with 5-minute breaks
- Start, pause, and reset functionality
- Session counter

### Progress Dashboard
- Completed tasks count
- Total study time tracking
- Progress levels (Beginner → Intermediate → Advanced → Expert)

### Data Persistence
- All data stored in browser localStorage
- Information persists across browser sessions

###  Dark Mode
- Toggle between light and dark themes
- Preference saved in localStorage

## Technologies Used

- **HTML** – Structure of the website
- **CSS** – Styling and responsive layout design
- **JavaScript** – Application logic and interactivity
- **LocalStorage** – Client-side data persistence

## How to Use

1. **Add Tasks**: Use the task planner to add study tasks with deadlines
2. **Get Suggestions**: Click "What Should I Study Now?" for smart recommendations
3. **Focus Timer**: Use the Pomodoro timer for productive study sessions
4. **Track Progress**: Monitor your study habits on the dashboard
5. **Dark Mode**: Toggle dark mode for comfortable viewing

## Getting Started

1. Clone or download the project files
2. Open `index.html` in a web browser
3. Start adding your study tasks!

For local development with a server:
```bash
cd braintrack
python3 -m http.server 8000
```
Then open `http://localhost:8000` in your browser.
